# 《奇怪的大冒险》

该项目是对移动端经典益智小游戏《奇怪的大冒险》进行的复刻，是笔者在学习Unity引擎2D系统时练手的小项目，因为时间原因目前只完成了第一阶段。不过基础框架已经搭建完毕，可以作为简单的框架参考。

## 第二阶段工作计划

1. 成就系统
   - 更过触发成就添加
2. 关卡系统
   - 实现未完成的关卡，包括地图、关卡控制器、游戏逻辑。
3. 音效系统
   - 音效文件分类
   - 在合适位置播放背景音乐
   - 在合适位置播放音效
4. 素材分类
   - 图片素材分类
   - 游戏素材分类
5. 游戏优化
   - 隐藏砖优化，防止侧边擦碰。（判断碰撞点法向量）
   - 考虑提示界面做成单独界面
   - 增加细节还原度
